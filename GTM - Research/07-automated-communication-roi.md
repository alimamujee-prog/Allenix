# Prompt 7: Medical Spa Automated Communication - ROI Rankings

Research the various automated communication tools and tactics available to medical spas, specifically **ranking them by ease of implementation vs revenue impact**.

## Communication Types to Analyze

1. **Appointment reminders** (SMS, email, phone)
2. **Pre-treatment instructions and prep**
3. **Post-treatment follow-up and check-ins**
4. **Re-booking and retention messages**
5. **Review requests after appointments**
6. **Abandoned booking recovery**
7. **Waitlist and cancellation fills**
8. **Birthday and milestone messages**
9. **Promotional campaigns**
10. **Lead response and inquiry handling**

## For Each Communication Type, Provide

### Implementation Metrics
- **Ease of Implementation:** How hard is it to set up (technical difficulty, integration needs)
- **Time to Setup:** Hours/days required
- **Monthly Cost:** Software or service costs

### Performance Metrics
- **Revenue Impact:** How much additional revenue or cost savings it generates
- **Time to ROI:** How long to see results
- **Success Rate:** Typical conversion or improvement rates

### Risk Assessment
- **Risk Level:** What could go wrong
- **Common pitfalls:** Where implementations fail

## Ranking Framework

Create a **"bang for the buck" score** for each:
- Score = (Revenue Impact ÷ (Effort × Cost)) × 100
- Rank from highest ROI/effort ratio to lowest
- Identify which ones medical spas should implement first, second, third

## Visual Output

Create a matrix with:
- **X-axis:** Effort (Low → High)
- **Y-axis:** Impact (Low → High)
- Plot each communication type
- Highlight "Quadrant 1" opportunities (Low Effort, High Impact)

## Real Examples

Include:
- Case studies or examples of practices that implemented these
- Before/after metrics (conversion rates, revenue, time savings)
- Common pitfalls to avoid
- Which tools/vendors are best for each

## Research Focus

Look for:
- Practice management software features
- Third-party automation tools
- SMS/email marketing platforms
- AI chatbot solutions
- Industry case studies
- Vendor benchmark data

## Output Format

1. **Ranked table** of all 10 communication types
2. **Quadrant matrix** (Effort vs Impact)
3. **Detailed deep-dive** on top 5 opportunities
4. **Implementation roadmap** (what to do first, second, third)
5. **Vendor recommendations** for each top opportunity

## Priority

Focus on AI-powered and automated solutions. Include specific tools and vendors where relevant. Prioritize tactics that can be set up once and run with minimal ongoing effort.
