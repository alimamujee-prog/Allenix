## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Accent (teal) | `#04CCBA` | Only accent color |
| Accent hover | `#029e90` | Hover states |
| Accent light | `#9eeee8` | Focus rings, subtle highlights |
| Accent wash | `#edfaf9` | Card/callout backgrounds |
| Background | `#faf9f6` | Page background (warm off-white) |
| Card surface | `#f2f0eb` | Card surfaces |
| Border | `#d4d0c8` | Dividers, input borders |
| Text primary | `#0d0d0d` | Headlines, body |
| Text secondary | `#1a1a1a` | Subheadings, captions |

Rules: No pure white (#fff). Teal is the only accent. No other hues.
