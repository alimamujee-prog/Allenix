# Allenix 10-Year Blueprint — Infographic Prompts

Four prompts for generating phase infographic images via Nano Banana Pro. Reference the Allenix logo at `Allenix Logo.png` in the vault root.

---

## Global Brand Style (prepend to each prompt)

> Style: Dark premium infographic on near-black background (#06080a). Minimalist and editorial. No gradients on backgrounds. Teal accent color (#00c8b4) for all highlights, data points, and callouts. Secondary text in cool gray (#8a9bb0). Surface cards in dark charcoal (#0d1117) with subtle 1px borders (#1e2730). Typography: bold serif for titles (similar to Playfair Display), monospace for labels and data (similar to IBM Plex Mono), clean serif for body text. No emojis. No clip art. No generic stock imagery. Data-driven, investor-pitch quality. Thin hairline dividers. Generous white space. Each image is a single infographic "slide" designed for a 16:9 presentation format.

## Usage Notes

- Use the Allenix logo (`Allenix Logo.png` in vault root) as an image reference/attachment when generating
- Generate each prompt as a separate image (16:9 landscape, 1920x1080 or higher)
- The "Global Brand Style" block at the top should be prepended to each individual prompt
- If the AI struggles with data accuracy on the charts, generate the visual layout first and overlay the data points manually in Figma or Canva
- For best results with chart data, consider generating the background/layout via AI and adding the actual SVG curve in post-production
