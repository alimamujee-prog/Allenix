## Section 3: How It Works

### Section label:
```
HOW IT WORKS
```

### Section headline:
```
From first call to first result.
```

### Three-step process:

**Step 1**: `Deep Discovery`
```
We sit down with you and your team. We map every revenue workflow, every bottleneck, every place time or money is leaking. Two to four weeks. The output is a single document: every opportunity ranked by ROI, each with a dollar number attached.
```

**Step 2**: `Identify the Constraint`
```
Not ten problems. One. The single constraint that, if removed, unlocks the most growth. We quantify it. We prove it. Then we agree on the fix.
```

**Step 3**: `Resolve It`
```
We build the solution using the latest AI tools and workflows. We train your team. We stay until it runs without us. Then we move to the next constraint.
```

### Closing line:
```
This is not a transformation roadmap. It is one problem, one fix, one result. Then the next one.
```

### Soft CTA (after closing line):
```
Ready to find your constraint?
[Book 30 Minutes]
```

**Annotation**: Matches the $5K workshop positioning perfectly. "From first call to first result" is tangible. Three steps. No ambiguity. Nirov reads this and knows exactly what he's buying. "One problem, one fix, one result" is the rhythm he remembers. The soft CTA catches visitors who are convinced by the process but might not scroll to the final section.

## Voice Checklist (apply to every line)

- [ ] Short sentences? (Max 20 words preferred)
- [ ] Active voice? (No "is being" or "are designed to")
- [ ] Numbers over adjectives? ("$200K" not "significant")
- [ ] No em dashes? (Use periods or colons)
- [ ] No banned phrases? (journey, ecosystem, synergy, cutting-edge, leverage, seamless)
- [ ] Houston-specific? (Ion District, Ship Channel, Gulf South — not generic)
- [ ] Written for a 50-year-old founder-CEO, not a tech buyer?
- [ ] Would Nirov say these words?
